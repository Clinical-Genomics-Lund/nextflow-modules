Welcome To Centrum För Molekylär Diagnostik (CMD)!
====================================================================
Centrum För Molekylär Diagnostik (CMD), a part of laboratory medicine at `Region Skåne`_ , offers healthcare based in Next generation sequencing (NGS) technology. NGS allows unbiased identification of gene mutations that affect hereditary, cancers and infectiotious diseases. 

.. prereq::

   * Access to hopper and lennart servers at CMD
   * Coyote and Bjorn access
   * Senteion valid license
   * Memory requirement > 180 GB
   * Uses threads > 50
